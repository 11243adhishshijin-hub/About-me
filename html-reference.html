📘 HTML Attributes & Elements Reference

Quick cheat sheet of commonly used HTML tags and attributes.

🏗️ Basic Elements

<h1> to <h6> → Headings

<p> → Paragraph

<a href=""> → Link

<img src="" alt=""> → Image

<div> → Container

<span> → Inline container

✏️ Text Attributes

title → Tooltip text

style → Inline CSS

class → CSS class name

id → Unique identifier

🔗 Link Attributes

href → Link destination

target="_blank" → Open in new tab

rel="noopener" → Security for new tabs

🖼️ Image Attributes

src → Image path

alt → Alternative text

width → Image width

height → Image height

loading="lazy" → Delays loading

📩 Forms

<form> → Form container

action → Where form sends data

method="GET/POST" → Submission type

<input> types:

  • text
  • email
  • password
  • number
  • checkbox

🔘 Buttons

onclick → Runs JS function

disabled → Disables button

type="submit" → Form submit button

🎥 Media

<video controls> → Video player

<audio controls> → Audio player

autoplay → Auto play media

loop → Repeat media

muted → No sound
